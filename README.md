# nodeLayout

![nodeLayout](https://zkboxing.com/friday/static/media/connection.2be91f9c.png)

## For Graph Layout or Edit.

### How to use

```bash
yarn add nodeLayout
```
OR
```bash
npm install nodeLayout
```

### Init
```js
nodeLayout.init({
  id: "node-wrap",
  data: ["step1", "step2"] 
});
```
### Add
```js
nodeLayout.add('step');
```
